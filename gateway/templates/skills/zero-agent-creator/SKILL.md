# Agent Creator Tools

This skill provides tools for listing available options and creating agents.

## Tools

- `list_providers`: Show available AI providers
- `list_skills`: Show available skills that can be added to agents
- `list_mcps`: Show available MCP servers
- `create_agent`: Create a new agent with the specified configuration

## Usage

The agent-creator uses these tools to:
1. Show available options to the user
2. Collect agent details conversationally
3. Create the agent when all details are gathered
