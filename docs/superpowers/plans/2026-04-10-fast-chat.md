# Fast Chat Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `/chat` route with a fast, direct chat agent that skips intent analysis and the planning pipeline. Same tools and delegation, faster response.

**Architecture:** Backend adds `mode` field to Invoke message and `skip_intent_analysis` flag to runner. New chat-specific system prompt template. Frontend adds a `/chat` page with minimal UI. Current `/` route is completely untouched.

**Tech Stack:** Rust (Axum, tokio), TypeScript/React

```markdown
<mode>direct</mode>

You are a direct assistant. Respond conversationally and take action immediately.

<rules>
- Answer questions directly. No planning step for simple tasks.
- Use tools when needed — read files, run commands, edit code, search.
- For multi-step tasks (3+ steps), use update_plan to track progress.
- Delegate to specialist agents only when the task clearly needs expertise you don't have.
- Always call respond when done. Include artifacts for any files you created.
- Be concise. The user wants fast answers, not essays.
</rules>

<delegation>
When a task needs deep research, complex coding, or multi-agent coordination:
- Use delegate_to_agent to spawn a specialist
- Available agents: call list_agents() to see options
- Set parallel: true for independent tasks
- You can delegate and continue working — don't wait unless you need the result
</delegation>
```

- [ ] **Step 2: Create chat_protocol.md shard**

Create `gateway/templates/shards/chat_protocol.md`:

```markdown
<agent_loop>
Each turn:
1. Read the user's message or tool result
2. Decide: can I answer directly, or do I need a tool?
3. If direct: call respond with the answer
4. If tool needed: call the tool, wait for result, then continue
No single-action-per-turn restriction — use multiple tools if needed.
</agent_loop>

<first_turn>
On a new conversation:
1. set_session_title — concise title
2. Start working immediately — no recall, no planning, no ward selection unless needed
3. If the user mentions a project/ward, enter it: ward(action="use", name="...")
</first_turn>
```

```rust
/// Assemble a lean system prompt for fast chat mode.
/// Uses chat_instructions.md instead of INSTRUCTIONS.md.
/// Uses only chat_protocol + tooling_skills shards.
pub fn assemble_chat_prompt(config_dir: &Path) -> String {
    let mut parts = Vec::new();
    
    // 1. SOUL.md (same identity)
    let soul = load_or_create_config(config_dir, "SOUL.md", "soul_starter.md");
    parts.push(soul);
    
    // 2. Chat-specific instructions (NOT INSTRUCTIONS.md)
    let instructions = load_or_create_config(config_dir, "chat_instructions.md", "chat_instructions.md");
    parts.push(instructions);
    
    // 3. OS.md
    let os = load_or_create_os(config_dir);
    parts.push(os);
    
    // 4. Minimal shards: chat_protocol + tooling_skills only
    let shards_dir = config_dir.join("shards");
    for name in &["chat_protocol", "tooling_skills"] {
        let path = shards_dir.join(format!("{}.md", name));
        if path.exists() {
            if let Ok(content) = std::fs::read_to_string(&path) {
                parts.push(content);
            }
        } else if let Some(embedded) = Templates::get(&format!("shards/{}.md", name)) {
            let content = String::from_utf8_lossy(&embedded.data).to_string();
            std::fs::write(&path, &content).ok();
            parts.push(content);
        }
    }
    
    let prompt = parts.join("\n\n---\n\n");
    tracing::info!(chars = prompt.len(), "Assembled fast chat prompt");
    prompt
}
```

- [ ] **Step 1: Create fast-chat-hooks.ts**

Simplified version of mission-hooks.ts — no intelligence feed, no phase tracking, no intent analysis blocks. Just messages and tool calls.

```typescript
// apps/ui/src/features/chat/fast-chat-hooks.ts
import { useState, useCallback, useRef, useEffect } from "react";
import { getTransport } from "@/services/transport";
import type { UploadedFile } from "./ChatInput";

const ROOT_AGENT_ID = "root";
const WEB_SESSION_KEY = "zbot_fast_session_id";
const WEB_CONV_KEY = "zbot_fast_conv_id";

interface ChatMessage {
  id: string;
  role: "user" | "assistant" | "tool";
  content: string;
  timestamp: string;
  toolName?: string;
  toolInput?: string;
  toolOutput?: string;
  isError?: boolean;
}

interface UseFastChatResult {
  messages: ChatMessage[];
  status: "idle" | "running" | "error";
  sendMessage: (text: string, attachments?: UploadedFile[]) => Promise<void>;
  stopAgent: () => Promise<void>;
  startNewSession: () => void;
}

export function useFastChat(): UseFastChatResult {
  const [messages, setMessages] = useState<ChatMessage[]>([]);
  const [status, setStatus] = useState<"idle" | "running" | "error">("idle");
  const isSubmittingRef = useRef(false);
  
  // Session management
  const getSessionId = () => localStorage.getItem(WEB_SESSION_KEY);
  const getConvId = () => {
    let id = localStorage.getItem(WEB_CONV_KEY);
    if (!id) {
      id = `fast-${crypto.randomUUID()}`;
      localStorage.setItem(WEB_CONV_KEY, id);
    }
    return id;
  };

  const conversationId = getConvId();

  // Subscribe to WebSocket events for streaming
  useEffect(() => {
    let unsubscribe: (() => void) | null = null;

    async function setup() {
      const transport = await getTransport();
      unsubscribe = transport.subscribeConversation(conversationId, {
        scope: "all",
        onMessage: (msg: any) => {
          // Handle streaming events
          if (msg.type === "token" && msg.delta) {
            setMessages(prev => {
              const last = prev[prev.length - 1];
              if (last?.role === "assistant" && !last.toolName) {
                return [...prev.slice(0, -1), { ...last, content: last.content + msg.delta }];
              }
              return [...prev, {
                id: crypto.randomUUID(),
                role: "assistant",
                content: msg.delta,
                timestamp: new Date().toISOString(),
              }];
            });
          }
          if (msg.type === "tool_call") {
            setMessages(prev => [...prev, {
              id: msg.tool_call_id || crypto.randomUUID(),
              role: "tool",
              content: "",
              timestamp: new Date().toISOString(),
              toolName: msg.tool,
              toolInput: typeof msg.args === "string" ? msg.args : JSON.stringify(msg.args),
            }]);
          }
          if (msg.type === "tool_result") {
            setMessages(prev => {
              const idx = prev.findIndex(m => m.id === msg.tool_call_id);
              if (idx >= 0) {
                const updated = [...prev];
                updated[idx] = { ...updated[idx], toolOutput: msg.result, isError: !!msg.error };
                return updated;
              }
              return prev;
            });
          }
          if (msg.type === "agent_completed" || msg.type === "respond") {
            setStatus("idle");
          }
          if (msg.type === "error") {
            setStatus("error");
          }
          if (msg.type === "invoke_accepted" && msg.session_id) {
            localStorage.setItem(WEB_SESSION_KEY, msg.session_id);
          }
        },
      });
    }

    setup();
    return () => { unsubscribe?.(); };
  }, [conversationId]);

  const sendMessage = useCallback(async (text: string, attachments: UploadedFile[] = []) => {
    if (!text.trim() || isSubmittingRef.current) return;
    isSubmittingRef.current = true;

    // Add user message
    setMessages(prev => [...prev, {
      id: crypto.randomUUID(),
      role: "user",
      content: text.trim(),
      timestamp: new Date().toISOString(),
    }]);

    setStatus("running");

    try {
      const transport = await getTransport();
      const sessionId = getSessionId() ?? undefined;

      // Build message with attachment table
      let message = text.trim();
      if (attachments.length > 0) {
        const header = "| File | Type | Size | Path |";
        const sep = "|------|------|------|------|";
        const rows = attachments.map(a => {
          const sizeStr = a.size < 1024 ? `${a.size} B`
            : a.size < 1024 * 1024 ? `${(a.size / 1024).toFixed(1)} KB`
            : `${(a.size / (1024 * 1024)).toFixed(1)} MB`;
          return `| ${a.name} | ${a.mimeType} | ${sizeStr} | ${a.path} |`;
        }).join("\n");
        message = `${message}\n\n**Attached files:**\n${header}\n${sep}\n${rows}`;
      }

      await transport.executeAgent(ROOT_AGENT_ID, conversationId, message, sessionId, "fast");
    } catch (error) {
      console.error("[FastChat] Failed to send:", error);
      setStatus("error");
    } finally {
      isSubmittingRef.current = false;
    }
  }, [conversationId]);

  const stopAgent = useCallback(async () => {
    try {
      const transport = await getTransport();
      await transport.stopAgent(conversationId);
    } catch (error) {
      console.error("[FastChat] Failed to stop:", error);
    }
  }, [conversationId]);

  const startNewSession = useCallback(() => {
    localStorage.removeItem(WEB_SESSION_KEY);
    localStorage.removeItem(WEB_CONV_KEY);
    setMessages([]);
    setStatus("idle");
    window.location.reload();
  }, []);

  return { messages, status, sendMessage, stopAgent, startNewSession };
}
```

**Note to implementer:** The `subscribeConversation` API and event shapes may differ from what's shown above. Read the existing `mission-hooks.ts` to understand the actual subscription pattern, event types, and callback shapes. Adapt the hook accordingly. The key differences from mission-hooks:
- No blocks/phases — just flat message list
- No intelligence feed state
- Passes `mode: "fast"` to executeAgent
- Uses separate localStorage keys (`zbot_fast_*`) so sessions don't collide

- [ ] **Step 2: Create FastChat.tsx page**

```tsx
// apps/ui/src/features/chat/FastChat.tsx
import { useRef, useEffect } from "react";
import { Plus, Square } from "lucide-react";
import { ChatInput } from "./ChatInput";
import { useFastChat } from "./fast-chat-hooks";
import ReactMarkdown from "react-markdown";
import remarkGfm from "remark-gfm";

export function FastChat() {
  const { messages, status, sendMessage, stopAgent, startNewSession } = useFastChat();
  const scrollRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (scrollRef.current) {
      scrollRef.current.scrollTop = scrollRef.current.scrollHeight;
    }
  }, [messages.length]);

  return (
    <div className="fast-chat">
      <div className="fast-chat__header">
        <span className="fast-chat__title">Chat</span>
        <div className="fast-chat__actions">
          {status === "running" && (
            <button className="btn btn--ghost btn--sm" onClick={stopAgent} title="Stop">
              <Square size={14} />
            </button>
          )}
          <button className="btn btn--ghost btn--sm" onClick={startNewSession} title="New chat">
            <Plus size={14} />
          </button>
        </div>
      </div>

      <div className="fast-chat__messages" ref={scrollRef}>
        {messages.length === 0 && (
          <div className="fast-chat__empty">
            <p>Ask anything. Fast, direct responses.</p>
          </div>
        )}
        {messages.map((msg) => (
          <div key={msg.id} className={`fast-chat__msg fast-chat__msg--${msg.role}`}>
            {msg.role === "user" && (
              <div className="fast-chat__bubble fast-chat__bubble--user">
                {msg.content}
              </div>
            )}
            {msg.role === "assistant" && (
              <div className="fast-chat__bubble fast-chat__bubble--agent">
                <ReactMarkdown remarkPlugins={[remarkGfm]}>{msg.content}</ReactMarkdown>
              </div>
            )}
            {msg.role === "tool" && (
              <div className="fast-chat__tool">
                <div className="fast-chat__tool-name">
                  {msg.toolName}
                  {msg.isError && <span style={{ color: "var(--destructive)" }}> (error)</span>}
                </div>
                {msg.toolOutput && (
                  <pre className="fast-chat__tool-output">{
                    msg.toolOutput.length > 500
                      ? msg.toolOutput.slice(0, 250) + "\n...\n" + msg.toolOutput.slice(-250)
                      : msg.toolOutput
                  }</pre>
                )}
              </div>
            )}
          </div>
        ))}
        {status === "running" && (
          <div className="fast-chat__typing">
            <span className="loading-spinner" style={{ width: 16, height: 16 }} />
          </div>
        )}
      </div>

      <div className="fast-chat__input">
        <ChatInput onSend={sendMessage} disabled={status === "running"} />
      </div>
    </div>
  );
}
```
